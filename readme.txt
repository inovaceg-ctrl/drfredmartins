A modern, minimalist WordPress theme inspired by Cal AI's clean design and focus on productivity.

== Description ==

Cal AI Inspired is a clean, modern WordPress theme designed for blogs and businesses. With its minimalist aesthetic and focus on readability, it provides an optimal viewing experience across all devices.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
